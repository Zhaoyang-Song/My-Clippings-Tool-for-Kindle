## 项目说明

基于 Python 3 ，用于将 Kindle 标注笔记文件 `My Clippings.txt` 转换成 html 文件。fork 自 [kindleNote](https://github.com/cyang812/kindleNote)，重写生成的 html 样式为 [书伴网 Clippings Fere 工具](https://bookfere.com/tools#ClippingsFere)，使之适配移动端。预览网站 [kindle.502.li](https://kindle.502.li)

## 使用说明

将 `My Clippings.txt` 文件拉至根目录下后运行 `kindle.py`
